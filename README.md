# MISO - Sancochof - DevOps 202512

### Comenzando

Requisitos: Python 3.10 o superior

1. Crear un entorno virtual:

```sh
python3 -m venv env
```

2. Activar entorno:

```sh
source env/bin/activate
```

3. Instalar dependencias:

```sh
pip install -r requirements.txt
```

4. Levantar aplicación:

```sh
flask run
```
